# Stuart Little Simulator : RPi client
Python script to be executed on RPi of bot

Note: commands mentioned below are to be executed from the RPiClient folder
## Install dependencies
Install and update required Python3 packages using

`pip3 install RPi picamera io logging socketserver requests threading http Rach --user`

## Start bot
Start the bot using

`python3 main.py name-of-bot IP-of-web-server`
